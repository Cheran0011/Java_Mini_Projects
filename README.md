Here I completed 3 console level projects for improving my problem solving and technical skills and also it will be helping to understand about real world problems.

1)ATM Banking System:
---------------------
The ATM Banking System is a console-based application that simulates basic banking operations such as account balance inquiry, cash withdrawal, deposit, and PIN authentication. It provides a simple and secure way to understand core banking functionalities.

🚀 Features:
------------
User authentication using PIN
Balance inquiry
Cash withdrawal
Deposit money
Transaction history (optional)
Input validation and error handling

2) Online Food ordering system:
------------------------------
This project is designed to allow users to browse food items, add them to a cart, and place orders online. It simulates real-world food delivery platforms with basic functionalities.

🚀 Features:
------------
User registration and login
Browse menu items
Add/remove items from cart
Place orders
Order summary and billing
Search and filter food items
